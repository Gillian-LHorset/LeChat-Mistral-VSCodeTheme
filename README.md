# LeChat-Mistral-VSCodeTheme

## Transformer le code en fichier `.vsix`

Dans le dossier conenant le `package.json`

`npx @vscode/vsce package`

## Ajouter le thème à VSCode

1. Fênetre `Extensions`
2. Les trois petits points en haut à droite
3. `Install from VSIX`
4. Ajouter le fichier
5. Fênetre `Extensions`
6. Cliquer sur le thème
7. `Set Color Theme`
